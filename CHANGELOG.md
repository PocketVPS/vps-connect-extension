# Changelog

All notable changes to Pocket VPS extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-11-08

### Added
- Initial release
- JWT authentication support
- One-click vps toggle
- URL whitelist management with wildcard support
- Visual status badge (ON/OFF)
- Secure credential storage
- PAC script generation for selective proxying
- Automatic token validation
- Bypass list for local addresses
- Modern and intuitive UI

### Security
- JWT token-based authentication
- Encrypted storage for credentials
- Automatic token refresh
- Secure HTTPS support via CONNECT method

## [Unreleased]
